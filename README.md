# How should you compile?

Use Visual Studio 2022 (MSVC compiler), install C++ support, after that, from the website [https://lua.org], download the lua binaries, and place the lua folder in the root of the project, and also place a DLL (maybe you should watch this video) [https://youtu.be/4l5HdmPoynw?si=fIlYEZs1dtbahDZp]
